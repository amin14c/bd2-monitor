# BD² Monitor — Bipolar II Digital Phenotyping System

نظام مراقبة رقمية سريرية لاضطراب ثنائي القطب II
مبني على مفهوم Digital Phenotyping

## نبذة عن المشروع

أداة بحثية سريرية لمراقبة أنماط المرضى المصابين باضطراب
Bipolar II عبر تحليل البيانات اليومية باستخدام الذكاء الاصطناعي.

تحليل البيانات مبني على معايير DSM-5 عبر Gemini AI.
